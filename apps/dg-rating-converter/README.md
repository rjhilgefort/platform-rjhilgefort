# DG Rating Converter

Disc golf rating converter tool.

## Quick Start

Run this app in development:

```bash
npm run dev --filter=@repo/dg-rating-converter
```

Visit: http://localhost:3004

**Live URL**: https://dgratingconverter.com/

## Environment Variables

No environment variables required.
