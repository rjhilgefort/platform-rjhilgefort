# @repo/dg-rating-converter

## Getting Started

1. Install dependencies

```
npm install
```

2. Run the dev server

```
npm run dev
```

Or build for production:

```
npm run build
npm run start
```

3. Docker

```
docker build -t rjhilgefort/dg-rating-converter:latest .
docker push rjhilgefort/dg-rating-converter:latest
```
