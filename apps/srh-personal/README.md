# srh-personal

Personal resume/portfolio site for Stephanie Hilgefort.

**Live:** https://stephanie.hilgefort.me

## Quick Start

```bash
npm run dev --filter=@repo/srh-personal
```

Visit: http://localhost:3006

## Features

- Timeline-based career history
- Company logos with fallback initials
- Responsive design (mobile-first)
- Google Analytics integration
- Social sharing (OG image, Twitter cards)
